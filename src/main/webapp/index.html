Time as a Service

Example usage

GET /taas/now
{ "hours": 17, "minutes": 31, "seconds": 35 }

GET /taas/now?timezone=Europe/London
{ "hours": 1, "minutes": 31, "seconds": 35 }

Description

Get the current hours, minutes, and seconds in JSON.
By default, this will be in the timezone of the server, but you can optionally pass a ?timezone=:timezone query parameter to use a different timezone.
